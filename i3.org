#+FILETAGS: :i3:

* Key-bindings :keybinding:

** With Mod
- W :: tabbed layout
- E :: default layout
- S :: stacked layout
- A :: focus parent
- R :: resize mode
- T or Enter :: Term (custom)
- D :: dmenu
- F :: full screen
- H :: split horizontally
- V :: split vertically

** With Shift-Mod
- Q :: kill window
- E :: exit i3
- R :: restart i3
